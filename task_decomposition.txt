Legal Document AI Agent – Task Decomposition

1.User Input
The user provides a legal document by pasting text into the system.

2.Preprocessing
The input text is prepared and cleaned for processing by the AI model.

3.Explanation Module
The system uses a language model to convert complex legal text into simple and understandable English.

4.Summary Module
The system generates a concise summary highlighting the key points of the document.

5.Tool Integration (Tavily Search)
The system optionally uses an external search API to retrieve additional context or explanations related to the legal content.

6.LLM-as-Judge
The system evaluates the quality of the generated explanation based on clarity and simplicity.

7.Output Display
The final explanation, summary, and additional information are displayed to the user through the interface.
